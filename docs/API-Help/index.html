API Overview

This API provides functions to create and manage projects in RepoRocket. You can create themes, plugins, and custom workflows. Functions include project creation, theme editing and exporting, as well as plugin initialization and integration.

Plugins

Plugins are kept in the /plugins folder. Each plugin must have a base.py file that defines an init_plugin(main_app) function. This function is called by RepoRocket during startup, allowing your plugin to interact with the application â€“ for example, adding menu items, status messages, or custom tools.

For detailed instructions on writing plugins, see our plugin development guide below.

Plugin Development Guide

  1. Create a folder under /f:/Projects/RepoRocket/plugins for your plugin.
  2. Add a base.py file that defines init_plugin(main_app).
  3. Inside init_plugin, register new actions, status messages, or extend UI components.
  4. Test your plugin by restarting RepoRocket.

Themes

The Theme API allows you to customize UI elements of RepoRocket and RRCreator. In our theme editor, for each UI element you can click a button to bring up a color wheel, type in a hex code if desired, and confirm to set that color. This is integrated via custom functions that are accessible to themes.

Custom Functions

You can extend RepoRocket and RRCreator via custom functions. For example, the newly added select_color function (used in the theme editor) opens a dialog with a color wheel and hex code input. Use these functions in your own plugins or theme customization routines.
